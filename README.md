# Bike Sales Analysis Dashboard

## Project Overview

This project analyzes customer demographics and purchasing behavior using a Bike Sales dataset. The objective is to identify the factors that influence whether a customer purchases a bicycle through data cleaning, pivot tables, and an interactive Excel dashboard.

## Dataset

The dataset contains customer information such as:

- Gender
- Age
- Income
- Education
- Occupation
- Marital Status
- Number of Children
- Home Ownership
- Number of Cars
- Commute Distance
- Region
- Bike Purchase Status

## Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Dashboard Design
- Data Cleaning
- Data Analysis

## Dashboard Features

The dashboard allows users to analyze bike purchases based on:
- Average Income
- Customer Age
- Commute Distance
- Gender
- Marital Status
- Region

Interactive slicers make it easy to filter the data and explore different customer groups.
